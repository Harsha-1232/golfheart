## 🚀 Quick Start

```bash
# Install dependencies
npm install
cd server && npm install && cd ..

# Run app (frontend + backend)
npm run fullstack
```

> Opens at **http://localhost:5173** *(check terminal for exact port)*

## 🔧 Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Frontend   | React 18 + Vite                     |
| Backend    | Node.js + Express                   |
| Database   | MongoDB (In-Memory, zero setup)     |
| Auth       | JWT + bcryptjs                      |
| File Upload| Multer                              |
| Styling    | Plain CSS                           |
